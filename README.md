# Social Media Application

## Presequisites
- Node
- NPM

## Instructions:
```sh
$ npm install
```

```sh
$ npm start
# Or run with Nodemon
$ npm run dev

# Visit http://localhost:5000
```

## Technologies:
- HTML
- CSS (Bootstrap)
- NodeJS (Passport, Express, Mongoose, handlebars, bcrypt)
- MongoDB (Atlas Flavour)


## Features:
- Validation
- Encrypted passwords
- Profiles
- Friend Requests
